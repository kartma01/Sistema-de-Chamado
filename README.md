# Sistema de Chamados - Node.js

Projeto desenvolvido para gerenciamento de chamados internos, com foco em lógica de programação, organização de código e manipulação de dados.

## Funcionalidades

- Cadastro de usuários
- Abertura de chamados
- Consulta de chamados
- Organização das solicitações

## Tecnologias

- JavaScript
- Node.js

## Objetivo

Projeto criado para praticar conceitos de desenvolvimento backend utilizando JavaScript e Node.js.

## Como executar

npm install

node index.js
